This python script takes a string entered by the user and creates it as a scrolling banner in Conway's Game of Life that loops infinitely. 

TO RUN
Script requires Golly to run so you must first install Golly

In golly: file -> "run script" then select hello_tickler.py

In order for Golly to interact with Python, you may have to find your python(version number).dll file and copy its path into a text boxthat will pop up when you click run script. 

The python dll file will likely be in your local C drive -> users -> your user -> app data (hidden folder) -> Local -> Python
